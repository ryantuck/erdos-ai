### `answer()` usage

The original problem asks: "Is it true that $G(n) \to \infty$ for almost all $n$?" — this is a yes/no question ("Is it true that..."), so **Case 1** applies.

**Main theorem (`erdos_673`):**
- [ ] **`answer()` missing (Case 1).** The problem asks a yes/no question and has been solved affirmatively (Tao's observation, noted in the module docstring; category is `research solved`). Per the checklist (§6, Case 1), the theorem should use `answer(True) ↔ P`, not state `P` directly. Currently:
  ```lean
  theorem erdos_673 :
      ∀ M : ℝ, HasNaturalDensityZero {n : ℕ | erdosG n ≤ M} := by
    sorry
  ```
  Should be:
  ```lean
  theorem erdos_673 :
      answer(True) ↔ ∀ M : ℝ, HasNaturalDensityZero {n : ℕ | erdosG n ≤ M} := by
    sorry
  ```
- [x] Quantifiers appear in the body of the type, not as theorem parameters (correct).

**Variant (`erdos_673.variants.average`):**
- [x] The variant formalizes the "easy" result stated by Erdős: $(1/X) \sum_{n \leq X} G(n) \to \infty$. This is presented as a direct assertion ("Erdős writes it is 'easy' to prove..."), so **Case 3** applies — no `answer()` wrapper needed.
- [x] Theorem type `Tendsto ... atTop atTop` directly asserts the divergence.

**Note on the second question:** The original problem also asks "Can one prove an asymptotic formula for $\sum_{n \leq X} G(n)$?" This yes/no question is not directly formalized. The variant addresses the weaker result that the average diverges, not whether a full asymptotic formula exists. Consider adding a separate variant for the asymptotic formula question using `answer(sorry) ↔ P` (Case 1).
