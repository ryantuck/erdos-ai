### `answer()` usage

The original problem asks: "Is it true that, in any two-colouring of the edges of $K_n$, there exist at most $\lceil\sqrt{n}\rceil$ monochromatic paths, all of the same colour, which together cover all vertices?" This is a yes/no question ("Is it true that..."), so **Case 1** applies.

The problem has been solved in the affirmative by Pokrovskiy, Versteegen, and Williams [PVW24], so the theorem should use `answer(True) ↔ P`.

- [ ] **FAIL**: Theorem type should be `answer(True) ↔ P` (Case 1, solved affirmatively), but is instead a direct assertion `∃ ...` (Case 3 style). The original problem phrasing is "Is it true that..." which triggers Case 1 per the checklist rules.
- [ ] **FAIL**: Quantifiers `(n : ℕ)`, `(c : Fin n → Fin n → Bool)`, and `(hc : ...)` are theorem **parameters** instead of appearing **after** `answer()` inside the RHS of `↔`. The checklist and linter `linter.style.answer_attribute` require quantifiers go after `answer()`.

The correct form should be:
```lean
theorem erdos_518 : answer(True) ↔
    ∀ (n : ℕ) (c : Fin n → Fin n → Bool),
      (∀ i j : Fin n, c i j = c j i) →
      ∃ (b : Bool) (paths : List (List (Fin n))),
        paths.length ≤ Nat.sqrt n + 1 ∧
        (∀ p ∈ paths, p.Nodup ∧ IsMonochromaticPath c b p) ∧
        (∀ v : Fin n, ∃ p ∈ paths, v ∈ p) := by sorry
```
