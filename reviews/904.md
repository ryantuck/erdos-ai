## Mathematical Correctness

The formalization accurately captures the Bollobás-Erdős conjecture (Edwards' theorem):

- **Hypothesis:** `(n : ℝ) ^ 2 / 4 < (G.edgeFinset.card : ℝ)` correctly formalizes "more than n²/4 edges" using strict inequality on reals.
- **Conclusion:** The existence of a triangle `x, y, z` with `G.Adj x y ∧ G.Adj y z ∧ G.Adj x z` and `(G.degree x + G.degree y + G.degree z : ℝ) ≥ 3 / 2 * (n : ℝ)` correctly formalizes the degree-sum condition. Note `3 / 2` on `ℝ` evaluates to 1.5, so `3 / 2 * n = 3n/2` as intended.
- **`DecidableRel G.Adj`:** Needed for `G.edgeFinset` and `G.degree` to be computable. Correctly placed as an instance parameter within the universal quantifier.
