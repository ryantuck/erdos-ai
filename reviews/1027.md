### `answer()` usage

The original problem asks: *"Must there exist ≫_c 2^{|X|} many sets B ⊂ X which intersect every set in F, yet contain none of them?"* — this is a **yes/no question** ("Must there exist..."), falling under **Case 1**.

The module docstring states it was "proved in the affirmative", so the expected form is:

```lean
theorem erdos_1027 : answer(True) ↔ P := by sorry
```

- [ ] **FAIL:** The theorem is stated as a direct assertion (Case 3 style) without `answer()`. Since the original problem is phrased as a yes/no question, it should use `answer(True) ↔ P` per checklist §6 and the formal-conjectures README §6. All quantifiers should appear on the RHS of the `↔`.

   ```lean
   theorem erdos_1027 :
       answer(True) ↔
       ∀ c : ℝ, c > 0 →
       ∃ δ : ℝ, δ > 0 ∧
       ∃ N₀ : ℕ, ∀ n : ℕ, n ≥ N₀ →
       ∀ F : Finset (Finset ℕ),
         (∀ A ∈ F, A.card = n) →
         (F.card : ℝ) ≤ c * (2 : ℝ) ^ n →
         let X := F.biUnion id
         ((X.powerset.filter (fun B =>
           (∀ A ∈ F, (A ∩ B).Nonempty) ∧
           (∀ A ∈ F, ¬(A ⊆ B)))).card : ℝ) ≥ δ * (2 : ℝ) ^ X.card := by
     sorry
   ```
