### `answer()` usage

- [ ] **ISSUE — Missing `answer()` wrapper.** The module docstring quotes the problem as asking "is it true that $h(n) \geq (2(\sqrt{3}-1) - o(1))n$?" This is a yes/no question (Case 1). Per the checklist, the theorem should be:
  ```lean
  theorem erdos_1033 : answer(sorry) ↔
      ∀ ε : ℝ, ε > 0 →
      ∃ N₀ : ℕ, ∀ n : ℕ, n ≥ N₀ →
      ∀ G : SimpleGraph (Fin n),
        G.edgeFinset.card > n ^ 2 / 4 →
        ∃ u v w : Fin n, G.Adj u v ∧ G.Adj v w ∧ G.Adj u w ∧
          (G.degree u + G.degree v + G.degree w : ℝ) ≥
            (2 * (Real.sqrt 3 - 1) - ε) * (n : ℝ) := by sorry
  ```
  Currently it is stated as a direct assertion (Case 3), but the English phrasing "is it true that" triggers Case 1.
