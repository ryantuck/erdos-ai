The original problem on erdosproblems.com poses three yes/no questions, all phrased as "Is it true that ...":

1. *"Is it true that, if ε > 0 and n is sufficiently large, whenever m ≤ (2 − ε)2^{n/2} the graph G_F has < 2^n many edges?"*
2. *"Is it true that if G_F has ≥ cm² edges then m ≪_c 2^{n/2}?"*
3. *"Is it true that, for any ε > 0, there exists some δ > 0 such that if there are > m^{2−δ} edges then m < (2+ε)^{n/2}?"*
