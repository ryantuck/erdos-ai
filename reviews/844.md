### `answer()` usage

- [ ] **FAIL.** The original problem at erdosproblems.com is phrased as a **yes/no question**: *"Is the maximum size of such an A achieved by taking A to be the set of even numbers and odd non-squarefree numbers?"* This is **Case 1**. Since the problem has been proved affirmatively (Weisenberg; Alexeev, Mixon, and Sawin [AMS25]), the theorem should use `answer(True) ↔ P`:
  ```lean
  theorem erdos_844 :
      answer(True) ↔ ∀ (N : ℕ) (A : Finset ℕ),
        A ⊆ Finset.Icc 1 N →
        (∀ a ∈ A, ∀ b ∈ A, ¬Squarefree (a * b)) →
        A.card ≤ (extremalSet N).card := by
    sorry
  ```
  Currently the theorem is stated as a direct assertion (Case 3 style) without `answer()`, which does not match the question-form phrasing of the original problem.

- [ ] **FAIL.** Quantifiers `(N : ℕ) (A : Finset ℕ)` are theorem **parameters** instead of appearing after `answer()`. Per §6 common rules, quantifiers must go **after** `answer(sorry)`, not as theorem parameters. The linter `linter.style.answer_attribute` enforces this.
