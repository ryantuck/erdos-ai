The original problem is a **direct assertion** (Case 3): "If G is a graph with 4k vertices and
minimum degree at least 2k then G contains k vertex-disjoint 4-cycles." It is not a yes/no
question and not a "find the value" question.

- [x] The theorem type is `P` directly (no `answer()` wrapper). Correct for Case 3.
- [x] Quantifiers are in the theorem body (using `∀`), not as theorem parameters. Correct.

**Formalization accuracy:** The source at erdosproblems.com/577 states: "If G is a graph with 4k
vertices and minimum degree at least 2k then G contains k vertex-disjoint 4-cycles." The
formalization correctly captures:
- Vertex set `Fin (4 * k)` for 4k vertices.
- Minimum degree condition `∀ v, 2 * k ≤ G.degree v`.
- Existence of `k` vertex-disjoint 4-cycles via `Fin k → ...` with `IsFourCycle` and `Disjoint`.
- `IsFourCycle` correctly requires pairwise distinct vertices and cyclic adjacency a~b~c~d~a.
- `fourCycleVertices` correctly extracts the set {a, b, c, d} for disjointness checking.
