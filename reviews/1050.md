### `answer()` usage

- [ ] **ISSUE: Missing `answer()` wrapper.** The original problem on erdosproblems.com asks: *"Is $\sum_{n=1}^{\infty} \frac{1}{2^n-3}$ irrational?"* — this is a **yes/no question** (Case 1). The problem has been solved affirmatively (by Borwein [Bo91]). Per the checklist, the theorem should use the Case 1 form with `answer(True)`:
  ```lean
  theorem erdos_1050 :
      answer(True) ↔ Irrational (∑' (n : ℕ), (1 : ℝ) / ((2 : ℝ) ^ (n + 1) - 3)) := by
    sorry
  ```
  The current formulation uses Case 3 (direct assertion) instead:
  ```lean
  theorem erdos_1050 :
      Irrational (∑' (n : ℕ), (1 : ℝ) / ((2 : ℝ) ^ (n + 1) - 3)) := by
    sorry
  ```
  The checklist states: *"The rule depends on how the **original problem is phrased in English**."* Since the original is "Is ... irrational?", Case 1 applies.

## Mathematical Correctness Note

The Lean formalization sums `∑' (n : ℕ), 1 / (2 ^ (n + 1) - 3)` over all `n : ℕ` (starting from 0). The shift `n + 1` correctly maps `n = 0, 1, 2, ...` to `2^1 - 3, 2^2 - 3, 2^3 - 3, ...`, matching the original sum $\sum_{n=1}^{\infty} \frac{1}{2^n - 3}$. No denominator is zero since $2^{n+1} = 3$ has no natural number solution.
