- [ ] **FAIL — wrong case pattern.** The original problem on erdosproblems.com is phrased as a yes/no question: *"Is it true that $f_k(n) \to \infty$ as $n \to \infty$?"* This is **Case 1** ("Is it true that …?"). Since the problem was **disproved**, the theorem should use `answer(False) ↔ P`, not a direct assertion.

  **Current (Case 3 — direct assertion of the Rödl–Tuza result):**
  ```lean

        fk k n = Nat.choose (k - 1) 2 := by

  ```

  **Expected (Case 1 — yes/no question, disproved):**
  ```lean
  theorem erdos_744 : answer(False) ↔

    sorry
  ```

  Alternatively, if the intent is to formalize the stronger Rödl–Tuza result rather than the yes/no question, the theorem should still wrap the assertion under `answer()` per Case 2 or be restructured to match the original problem's phrasing.

### Note on mathematical content

The mathematical content is well-crafted. The three helper definitions are clean and well-documented:

- `IsKCritical` correctly captures $k$-criticality (colorable in $k$ colors, not in $k{-}1$, and every proper subgraph is $(k{-}1)$-colorable).
- `minEdgesToBipartite` correctly defines the minimum number of edge deletions to achieve 2-colorability via `sInf` over finsets of edges.
- `fk` correctly defines the minimum of `minEdgesToBipartite` over all $k$-critical graphs on $n$ vertices.

The theorem correctly formalizes the Rödl–Tuza result: for every $k \ge 3$, there exists $N_0$ such that for all $n \ge N_0$, $f_k(n) = \binom{k-1}{2}$. Note the theorem uses $k \ge 3$ which is stronger than the original problem's $k \ge 4$; this is mathematically correct since the result holds more generally.
