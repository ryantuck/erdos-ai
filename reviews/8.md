### `answer()` usage

- **ISSUE**: The original problem is phrased as a yes/no question. The module docstring states: *"The original Erdős–Graham conjecture **asked**: for any finite colouring of the positive integers, **must there exist** a covering system all of whose moduli are monochromatic?"* This matches the checklist triggers for Case 1 ("asked whether", "must there exist"). Per the checklist, the theorem should use the `answer() ↔ P` form:

  ```lean
  theorem erdos_8 : answer(False) ↔
      (∀ k : ℕ, 0 < k → ∀ χ : ℕ → Fin k,
        ∃ S : Finset (ℤ × ℕ), IsCoveringSystem S ∧ HasMonochromaticModuli χ S) := by
    sorry
  ```

  where `P` formalizes the question as originally asked (affirmatively: "there must exist a monochromatic covering system for every colouring") and `answer(False)` encodes that the answer is "no".

  Currently the theorem is stated as a direct existential assertion of the negation (Case 3 style):

  ```lean
  theorem erdos_8 :
      ∃ k : ℕ, 0 < k ∧ ∃ χ : ℕ → Fin k,
        ∀ S : Finset (ℤ × ℕ), IsCoveringSystem S → ¬HasMonochromaticModuli χ S := by
    sorry
  ```

  This formulation is mathematically equivalent but does not conform to the checklist's `answer()` convention for yes/no problems.
