### `answer()` usage — **FAIL**

The original problem on erdosproblems.com is phrased as a yes/no question:

> *"Is it true that in any 2-colouring of the edges of $K_n$ there must exist at least $(1+o(1))n^2/12$ many edge-disjoint monochromatic triangles?"*

This is **Case 1** ("Is it true that..."). Since the problem is solved affirmatively (proved by Gruslys and Letzter [GrLe20]), the theorem should use `answer(True) ↔ P`:

```lean
theorem erdos_76 : answer(True) ↔
    ∀ ε : ℝ, 0 < ε →
    ∃ N : ℕ, ∀ n : ℕ, N ≤ n →
    ∀ col : EdgeTwoColoring n,
    ∃ 𝒯 : Finset (Finset (Fin n)),
      (∀ T ∈ 𝒯, IsMonochromaticTriangle col T) ∧
      IsEdgeDisjointFamily 𝒯 ∧
      (1 - ε) * (n : ℝ) ^ 2 / 12 ≤ (𝒯.card : ℝ) := by
  sorry
```

**Current formulation** uses Case 3 (direct assertion without `answer()`), which is incorrect for a yes/no question.

- [ ] Should use `answer(True) ↔ P` (Case 1, solved affirmatively) — currently uses bare `P` (Case 3).
- [x] Quantifiers are in the theorem body, not as parameters (correct regardless of case).
