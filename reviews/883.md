### `answer()` usage

The original problem on erdosproblems.com is phrased as a question that Erdős and Sárközy "asked whether" the threshold guarantees all odd cycles of bounded length. "Asked whether" is explicitly listed in the checklist as a **yes/no question** trigger (Case 1).

- [ ] **ISSUE:** The theorem is formalized as a direct assertion (Case 3):
  ```lean
  theorem erdos_883 : ∀ n : ℕ, ∀ A : Finset ℕ, ... → ... → ...
  ```
  Since the source problem is phrased as a yes/no question ("asked whether"), Case 1 should apply:
  ```lean
  theorem erdos_883 : answer(sorry) ↔
      ∀ n : ℕ, ∀ A : Finset ℕ,
        (∀ a ∈ A, 1 ≤ a ∧ a ≤ n) →
        A.card > erdos883Threshold n →
        ∀ k : ℕ, k ≥ 3 → k % 2 = 1 → k ≤ n / 3 + 1 →
          (coprimeGraph A).ContainsCycle k := by
    sorry
  ```
  **Mitigating factor:** Erdős and Sárközy already proved the result for cycles of length ≤ cn (some c > 0), so the conjecture is widely believed to be true and the open question is about the sharp bound. One could argue the problem is effectively asserting the conjecture. However, the checklist rules are explicit: if the source says "asked whether", use `answer(sorry) ↔ P`.

### Multi-part coverage

- [ ] **ISSUE:** The original problem has **two parts**: (1) all odd cycles of length ≤ n/3 + 1, and (2) a complete (1,ℓ,ℓ) tripartite subgraph on 2ℓ+1 vertices for sufficiently large n. Only part (1) is formalized. The second part should either be formalized as a variant (`erdos_883.variants.tripartite`) or the omission should be documented in the module docstring.

1. **Module docstring missing problem description (§3):** The module docstring has a bibliography entry but no plain-English summary of the problem. Add a brief description of the conjecture.
