**Note on `Collinear3`:** The definition checks that difference vectors `q - p` and `r - p` are ℤ-linearly dependent via `∃ a b : ℤ, (a ≠ 0 ∨ b ≠ 0) ∧ ∀ k, a * (q k - p k) = b * (r k - p k)`. This is mathematically equivalent to the standard collinearity condition (linear dependence over ℤ implies linear dependence over ℚ and vice versa for integer vectors). The definition is correct. One could consider using Mathlib's `Collinear` from `LinearAlgebra.AffineSpace.Independent`, but the custom definition is simple, self-contained, and well-documented.
