## Formalization Faithfulness

The formalization correctly captures the original Erdős–Simonovits conjecture:

- **Bipartiteness** is encoded via `Nonempty (H.Coloring (Fin 2))` (2-colorability). ✓
- **Minimum degree ≥ r** is encoded via `∀ v : U, r ≤ H.degree v`. ✓
- **Turán number** `turanNumber H n` is defined as the supremum of edge counts over H-free graphs on n vertices. ✓
- **The asymptotic lower bound** ex(n; H) ≫ n^{2−1/(r−1)+ε} is correctly formalized as ∃ ε > 0, C > 0, N₀, ∀ n ≥ N₀, C · n^{2−1/(r−1)+ε} ≤ ex(n; H). ✓
- **Disproof**: `answer(False)` correctly reflects Janzer's disproof for r = 3 (constructing a 3-regular bipartite H with ex(n; H) ≪ n^{4/3+δ}, contradicting the n^{3/2+ε} lower bound the conjecture would require). ✓

Note: The formalization quantifies over all r ≥ 2 and all H with every vertex having degree ≥ r. The original conjecture uses "minimum degree r" (exactly r). These are equivalent in truth value: the disproof works by exhibiting an r-regular graph (where min degree equals r), and the formalization's hypothesis is satisfied since every vertex has degree ≥ r.
