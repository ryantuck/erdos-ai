### `answer()` usage

- [ ] **Issue — Case 1 should apply but `answer()` is absent.** The original problem at erdosproblems.com asks: *"In particular, does $\lim_{d \to \infty} g_d(n)/d^{n-1}$ exist?"* The phrasing "does ... exist?" is a yes/no question, which is an explicit Case 1 trigger per the checklist. Since the problem has been resolved (the limit exists and equals $1/(n-1)!$), the theorem should use `answer(True) ↔ P` format, e.g.:

  ```lean
  theorem erdos_1089 :
      answer(True) ↔
        ∀ (n : ℕ), 2 ≤ n →
          Tendsto (fun d : ℕ => (g d n : ℝ) / (d : ℝ) ^ (n - 1))
            atTop (nhds (1 / (Nat.factorial (n - 1) : ℝ))) := by sorry
  ```

  Currently the theorem states the limit as a direct assertion (Case 3), which does not match the yes/no phrasing of the source problem.

- [ ] **Quantifiers should go after `answer(...)`, not as theorem parameters.** Currently `(n : ℕ) (hn : 2 ≤ n)` are theorem parameters. If `answer()` is added (Case 1), these must move to the RHS of `↔` as `∀ (n : ℕ), 2 ≤ n → ...`.
