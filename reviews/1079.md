### `answer()` usage

- [ ] **ISSUE — Missing `answer()` wrapper.** The original problem asks a yes/no question: *"must G contain a vertex with degree d ≫_r n whose neighbourhood contains at least ex(d; K_{r-1}) edges?"* Per the checklist, this is **Case 1** (yes/no question). Since the problem is solved affirmatively (Bollobás–Thomason [BoTh81]), the theorem should use:
  ```lean
  theorem erdos_1079 : answer(True) ↔ P := by sorry
  ```
  Currently formalized as a **Case 3** direct assertion (`P := by sorry`).

- [ ] **ISSUE — Quantifiers as theorem parameters.** `r : ℕ` and `hr : r ≥ 4` are theorem parameters. If `answer()` is used, quantifiers must go **after** `answer()`, not as parameters:
  ```lean
  -- Current (non-conforming):
  theorem erdos_1079 (r : ℕ) (hr : r ≥ 4) : ∃ c : ℝ, ... := by sorry
  -- Expected:
  theorem erdos_1079 : answer(True) ↔ ∀ r : ℕ, r ≥ 4 → ∃ c : ℝ, ... := by sorry
  ```
