### `answer()` usage

- [ ] **Issue — Missing `answer()` wrapper (Case 1 violation).** The original problem asks a yes/no question: *"Is it true that, for all sufficiently large n, there exists some i < n such that p_n² < p_{n+i} · p_{n−i}?"* Per the checklist (Case 1), since the answer is no, the main theorem should be:
  ```lean
  theorem erdos_453 : answer(False) ↔
      (∀ᶠ n in Filter.atTop, ∃ i : ℕ, 0 < i ∧ i < n ∧
        nthPrime n ^ 2 < nthPrime (n + i) * nthPrime (n - i)) := by sorry
  ```
  The current formulation instead directly states Pomerance's result (the stronger negative counterexample) without using `answer()` at all. While the mathematics is correct — and the current statement is a valid and informative formalization of the disproof — it does not conform to the Case 1 `answer(False) ↔ P` pattern required when the original problem is phrased as "Is it true that...".

  **Suggested fix:** Reformulate the main theorem with `answer(False) ↔ P` where `P` is the original conjecture. The current Pomerance result could be kept as a variant (`erdos_453.variants.pomerance`) or as a supporting lemma.
