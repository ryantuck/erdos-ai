### Mathematical notes

The formalization is mathematically reasonable:
- **Entireness** is captured by `Differentiable ℂ f` (complex-differentiable everywhere on ℂ).
- **Non-linearity** is captured by `¬∃ a b : ℂ, ∀ z, f z = a * z + b` (f is not affine). This is slightly weaker than "transcendental" (the Barth–Schneider result), but faithfully represents the original problem's requirement.
- **Rationality preservation** is captured by the biconditional on real inputs: `(∃ q : ℚ, (q : ℝ) = x) ↔ ∃ q : ℚ, (q : ℂ) = f ↑x`. Note this embeds ℚ into ℂ via the canonical path ℚ → ℝ → ℂ, so `f(x) ∈ ℚ` means f(x) is a rational real number viewed in ℂ.
