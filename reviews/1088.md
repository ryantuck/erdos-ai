### `answer()` usage

- [ ] **Issue — Case 1 should apply but `answer()` is absent.** The original problem (both at erdosproblems.com and in the module docstring) asks: *"is it true that, for fixed $n \geq 3$, $f_d(n) = 2^{o(d)}$?"* The phrasing "is it true that" is an explicit Case 1 trigger per the checklist. The theorem should use `answer(sorry) ↔ P` format, e.g.:

  ```lean
  theorem erdos_1088 :
      answer(sorry) ↔
        ∀ (n : ℕ), n ≥ 3 →
          ∀ ε : ℝ, ε > 0 →
            ∃ D₀ : ℕ, ∀ d : ℕ, d ≥ D₀ →
              (erdos_f d n : ℝ) ≤ (2 : ℝ) ^ (ε * (d : ℝ)) := by sorry
  ```

  Currently the theorem states the bound as a direct assertion (Case 3), which does not match the yes/no phrasing of the source problem.

- [ ] **Quantifiers should go after `answer(sorry)`, not as theorem parameters.** Currently `(n : ℕ) (hn : n ≥ 3)` are theorem parameters. If `answer()` is added (Case 1), these must move to the RHS of `↔` as `∀ (n : ℕ), n ≥ 3 → ...`.
