### `answer()` usage

The original problem on [erdosproblems.com/79](https://www.erdosproblems.com/79) is phrased as a **yes/no question**: *"Are there infinitely many graphs G which are not Ramsey size linear but such that all of their proper subgraphs are?"* The module docstring confirms this phrasing. This is **Case 1**.

The theorem currently uses:

```lean
theorem erdos_79 : answer(True) ↔
    ∀ N : ℕ, ∃ (p : ℕ) (G : SimpleGraph (Fin p)),
      p ≥ N ∧
      ¬ IsRamseySizeLinear G ∧
      ∀ (q : ℕ) (H : SimpleGraph (Fin q)),
        IsSubgraphOf H G →
        ¬ IsSubgraphOf G H →
        IsRamseySizeLinear H := by
  sorry
```

- [x] Quantifiers go **after** `answer(True)`, not as theorem parameters: the `∀ N` and all quantifiers appear inside the RHS of the `↔`.
