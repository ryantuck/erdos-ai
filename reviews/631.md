### `answer()` usage

The original problem at erdosproblems.com/631 is phrased as **two yes/no questions**:

1. *"Does every planar graph G have χ_L(G) ≤ 5?"*
2. *"Is this best possible?"*

This is **Case 1** (yes/no question). Both questions have been answered affirmatively.

- [ ] **Main theorem `erdos_631` should use `answer(True) ↔ P`.** ❌ Currently written as a direct assertion (Case 3 style) with universally quantified parameters:
  ```lean
  theorem erdos_631 {V : Type*} [Fintype V] [DecidableEq V]
      (G : SimpleGraph V) (hplanar : IsPlanar G) :
      IsChoosable G 5 := by sorry
  ```
  Should be:
  ```lean
  theorem erdos_631 : answer(True) ↔
      ∀ (V : Type*) [Fintype V] [DecidableEq V]
        (G : SimpleGraph V), IsPlanar G → IsChoosable G 5 := by sorry
  ```

- [ ] **Variant `erdos_631.variants.tight` should use `answer(True) ↔ P`.** ❌ Currently:
  ```lean
  theorem erdos_631.variants.tight :
      ∃ (V : Type) (_ : Fintype V) (G : SimpleGraph V),
        IsPlanar G ∧ ¬IsChoosable G 4 := by sorry
  ```
  Should be:
  ```lean
  theorem erdos_631.variants.tight : answer(True) ↔
      ∃ (V : Type) (_ : Fintype V) (G : SimpleGraph V),
        IsPlanar G ∧ ¬IsChoosable G 4 := by sorry
  ```

- [ ] **Quantifiers go after `answer()`, not as theorem parameters.** ❌ `erdos_631` has `{V : Type*} [Fintype V] [DecidableEq V] (G : SimpleGraph V) (hplanar : IsPlanar G)` as parameters. Under Case 1, all quantifiers must be on the RHS of `↔`.

**Verdict: FAIL** — Both theorems use Case 3 form but the original problem is phrased as yes/no questions (Case 1). Must use `answer(True) ↔ P` with quantifiers on the RHS.
