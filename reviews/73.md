Since the problem has been solved affirmatively (Reed [Re99]), the correct formulation should be:

```lean
theorem erdos_73 : answer(True) ↔
    ∀ k : ℕ, ∃ C : ℕ,

        (∀ S : Finset (Fin n), ∃ I : Finset (Fin n), G.IndepSetIn I S ∧
          2 * I.card ≥ S.card - k) →
        ∃ T : Finset (Fin n), T.card ≤ C ∧
```
