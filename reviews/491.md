### `answer()` usage

The original problem on erdosproblems.com asks: *"If there is a constant $c$ such that $|f(n+1)-f(n)| < c$ for all $n$ then **must** there exist some $c'$ such that $f(n) = c'\log n + O(1)$?"* — this is a yes/no question ("must there exist...?"), so **Case 1** applies. The problem has been proved affirmatively by Wirsing [Wi70].

- [ ] **FAIL:** The theorem uses Case 3 format (direct assertion with no `answer()` wrapper), but the original problem is phrased as a yes/no question, requiring Case 1: `answer(True) ↔ P`. The correct form should be:
  ```lean
  theorem erdos_491 : answer(True) ↔
      ∀ f : ℕ → ℝ, IsAdditiveArithmeticFunction f →
      (∃ c : ℝ, ∀ n : ℕ, |f (n + 1) - f n| < c) →
      ∃ c' : ℝ, ∃ M : ℝ, ∀ n : ℕ, 1 ≤ n →
        |f n - c' * Real.log (n : ℝ)| ≤ M := by sorry
  ```
- [ ] **FAIL:** Quantifiers (`f`, `hf`, `hbdd`) are theorem parameters instead of appearing after `answer()` inside the RHS of `↔`. The checklist and linter `linter.style.answer_attribute` require quantifiers to go after `answer(sorry)`, not as theorem parameters.
