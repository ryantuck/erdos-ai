### `answer()` usage

- [ ] **VIOLATION — Missing `answer()` wrapper.** The original problem asks a yes/no question: "is there such a graph for $g(n) = (\log n)^3$?" Per the checklist §6 Case 1, the main theorem should use `answer(sorry) ↔ P` form. The current formalization directly asserts non-existence (Case 3 style) without `answer()`.

  **Current (Case 3 style):**
  ```lean
  theorem erdos_805 :
      ∃ N₀ : ℕ, ∀ n : ℕ, n ≥ N₀ →
        ¬∃ G : SimpleGraph (Fin n), ErdosHajnalProperty G
          (Nat.floor ((Real.log (n : ℝ)) ^ 3))
          (Nat.ceil (Real.log (n : ℝ))) := by sorry
  ```

  **Expected (Case 1 style):**
  ```lean
  theorem erdos_805 : answer(sorry) ↔
      ∀ n : ℕ, n ≥ 2 →   -- or ∃ N₀, ∀ n ≥ N₀ →
        ∃ G : SimpleGraph (Fin n), ErdosHajnalProperty G
          (Nat.floor ((Real.log (n : ℝ)) ^ 3))
          (Nat.ceil (Real.log (n : ℝ))) := by sorry
  ```
  where the RHS formalizes the question *as asked* ("is there such a graph?"), and `answer(False)` would mean "no".

**Note on docstring accuracy:** The docstring for `erdos_805.variants.alon_sudakov` says "This strengthens the Erdős–Hajnal conjecture." This is potentially misleading. The Alon–Sudakov result proves non-existence for the *smaller* function $g(n) = c(\log n)^3/\log\log n$, which is a *weaker* result than the conjecture (non-existence for larger $g$ implies non-existence for smaller $g$, not vice versa). It would be more accurate to say "This is a partial result toward the Erdős–Hajnal conjecture" or "This proves a weaker form of the conjecture."
