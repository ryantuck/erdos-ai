### `answer()` usage

The module docstring states: *"Must there be two distances which occur at least once but between at most n pairs of points? Must the number of such distances tend to infinity as n → ∞?"* — both are yes/no questions (**Case 1**).

- [ ] **FAIL**: `erdos_132` should use `answer(sorry) ↔ P` form, but currently states `P` directly (Case 3 style). The problem asks "Must there be…?" which is a yes/no question. Should be:
  ```lean
  theorem erdos_132 :
      answer(sorry) ↔
        ∀ A : Finset (EuclideanSpace ℝ (Fin 2)), 5 ≤ A.card →
          2 ≤ Set.ncard (limitedOccurrences A) := by
    sorry
  ```
- [ ] **FAIL**: `erdos_132.variants.tend_to_infinity` has the same issue. Should be:
  ```lean
  theorem erdos_132.variants.tend_to_infinity :
      answer(sorry) ↔
        ∀ k : ℕ, ∃ N : ℕ, ∀ A : Finset (EuclideanSpace ℝ (Fin 2)), N ≤ A.card →
          k ≤ Set.ncard (limitedOccurrences A) := by
    sorry
  ```
- [x] Quantifiers are not theorem parameters (they are universally quantified in the body, which is correct).
