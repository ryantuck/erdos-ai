  The original problem at [erdosproblems.com/21](https://www.erdosproblems.com/21) asks:
  > "Is it true that $f(n) \ll n$?"

  This is a yes/no question ("Is it true that ..."), which falls under **Case 1** of the checklist. Since the problem has been solved affirmatively (by Kahn [Ka94]), the theorem should use:

  ```lean
  theorem erdos_21 : answer(True) ↔
      (∃ C : ℝ, 0 < C ∧ ∃ N : ℕ, ∀ n : ℕ, N ≤ n →
        (erdosLovaszF n : ℝ) ≤ C * n) := by
    sorry
  ```

  The current formalization uses Case 3 (direct assertion) instead:

  ```lean
  theorem erdos_21 :
      ∃ C : ℝ, 0 < C ∧ ∃ N : ℕ, ∀ n : ℕ, N ≤ n →
        (erdosLovaszF n : ℝ) ≤ C * n := by
    sorry
  ```

  Per the checklist: "If it asks a yes/no question — 'asked whether', 'is it true that', ... — use Case 1."
