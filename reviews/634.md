### `answer()` usage

- [ ] **Potential issue.** The module docstring says *"Find all n such that..."*, which is a **Case 2** ("Find / Determine") problem. Per the checklist, this should use `= answer(sorry)`, e.g.:
  ```lean
  theorem erdos_634 : {n : ℕ | HasCongruentDissection n} = answer(sorry) := by sorry
  ```
  Instead, the main theorem formalizes a specific **conjecture** about the problem as a direct assertion (Case 3):
  ```lean
  theorem erdos_634 :
      ∀ p : ℕ, Nat.Prime p → p % 4 = 3 → ¬ HasCongruentDissection p := by sorry
  ```
  **Justification for current form:** The full problem ("find all n") does not yet have a conjectured complete characterization. The main theorem captures the strongest known open conjecture about the problem. The theorem docstring describes it as a conjecture ("It is conjectured that..."), making Case 3 defensible. Reviewers should decide whether a Case 2 wrapper for the full problem is also needed.
