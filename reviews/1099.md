### `answer()` usage

- [ ] **FAIL.** The original problem asks *"Is it true that $\liminf h_\alpha(n) \ll_\alpha 1$?"* — this is a yes/no question ("Is it true that …"), so **Case 1** applies. The theorem should use the `answer() ↔ P` form. Since Vose proved it affirmatively, it should be `answer(True) ↔ P`.

  **Current (incorrect):**
  ```lean
  theorem erdos_1099 (α : ℝ) (hα : α > 1) :
      ∃ C : ℝ, ∀ N : ℕ, ∃ n : ℕ, n ≥ N ∧ hAlpha α n ≤ C := by
    sorry
  ```

  **Expected:**
  ```lean
  theorem erdos_1099 : answer(True) ↔
      ∀ α : ℝ, α > 1 → ∃ C : ℝ, ∀ N : ℕ, ∃ n : ℕ, n ≥ N ∧ hAlpha α n ≤ C := by
    sorry
  ```

- [ ] **FAIL.** Quantifiers `(α : ℝ) (hα : α > 1)` are theorem parameters. Per the checklist (and `linter.style.answer_attribute`), quantifiers must go **after** `answer(...)`, not as parameters. The corrected form above moves `∀ α : ℝ, α > 1 →` to the RHS of the `↔`.
