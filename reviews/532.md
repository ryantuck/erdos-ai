### `answer()` usage

- [ ] **FAIL.** The original problem is phrased as a yes/no question: *"If ℕ is 2-coloured then is there some infinite set A⊆ℕ such that all finite subset sums … are monochromatic?"* This is a **Case 1** problem ("is there …" / "does there exist"). Since it was proved affirmatively by Hindman, the theorem should use:
  ```lean
  theorem erdos_532 : answer(True) ↔
      ∀ χ : ℕ → Bool, ∃ (A : Set ℕ), A.Infinite ∧ (∀ a ∈ A, 0 < a) ∧
      ∃ c : Bool, ∀ S : Finset ℕ, S.Nonempty → (↑S : Set ℕ) ⊆ A →
        χ (∑ i ∈ S, i) = c := by sorry
  ```
  Currently the theorem is stated as a direct assertion (Case 3) with `χ` as a parameter, which violates two rules:
  1. Missing `answer(True) ↔` wrapper for a yes/no question.
  2. Quantifier `χ` appears as a theorem parameter instead of after `answer()`. The linter `linter.style.answer_attribute` would flag this once `answer()` is added.

## Summary

Per the checklist §6 Case 1, the theorem should be wrapped with `answer(True) ↔` and the colouring `χ` should be universally quantified inside the statement rather than passed as a parameter. Currently it uses Case 3 (direct assertion) form, which is incorrect for a question-phrased problem.
