### `answer()` usage

The original problem on erdosproblems.com is phrased as a yes/no question: *"Is it true that, if n is sufficiently large depending on C, then in any 2-colouring of {2,…,n} choose 2 there exists some X ⊆ {2,…,n} such that X choose 2 is monochromatic and ∑ 1/log x ≥ C?"* This is a clear "Is it true that …" phrasing, so **Case 1** applies.

The problem has been solved affirmatively (by Rödl [Ro03]), so the correct form is `answer(True) ↔ P`.

- [ ] **FAIL**: The theorem is stated as a direct assertion (Case 3 style) rather than using `answer(True) ↔ P` (Case 1). The current statement:
  ```lean
  theorem erdos_191 :
      ∀ C : ℝ, C > 0 →
        ∃ N : ℕ, ∀ n ≥ N, ...
  ```
  should instead be:
  ```lean
  theorem erdos_191 :
      answer(True) ↔
        ∀ C : ℝ, C > 0 →
          ∃ N : ℕ, ∀ n ≥ N, ...
  ```
- [x] Quantifiers are not theorem parameters (they are inside the statement body).
