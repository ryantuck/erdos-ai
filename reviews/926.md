### `answer()` usage

The original problem asks: **"Is it true that ex(n; H_k) ≪_k n^{3/2}?"** — this is a yes/no question ("Is it true that…"), which maps to **Case 1**. The problem has been solved affirmatively (Füredi 1991, improved by Alon–Krivelevich–Sudakov 2003), and the category is `research solved`.

- [ ] **Incorrect: theorem uses Case 3 style (direct assertion `P`) instead of Case 1.** Per the checklist, a yes/no question that has been solved affirmatively should be stated as:
  ```lean
  theorem erdos_926 : answer(True) ↔
      ∀ (k : ℕ), 4 ≤ k →
      ∃ C : ℝ, 0 < C ∧ ∀ n : ℕ, 1 ≤ n →
        (turanNumber (graphHk k) n : ℝ) ≤ C * (n : ℝ) ^ ((3 : ℝ) / 2) := by sorry
  ```
  The current formalization omits the `answer(True) ↔` wrapper entirely.
- [x] Quantifiers are inside the theorem type, not as parameters — would be correct once `answer()` is added.

### Mathematical accuracy

The formalization is mathematically sound:
- `turanNumber` correctly defines ex(n; H) as the supremum of edge counts over H-free simple graphs on n vertices.
- `graphHk` correctly constructs H_k: center vertex x (`Sum.inl ()`) adjacent to all spokes y_i (`Sum.inr (Sum.inl i)`), and each pair vertex z_{ij} (`Sum.inr (Sum.inr ⟨(i,j), _⟩)` with `i < j`) adjacent to y_i and y_j.
- The bound `≤ C * n^{3/2}` with C depending on k correctly formalizes `≪_k n^{3/2}`.
