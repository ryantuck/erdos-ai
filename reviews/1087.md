### `answer()` usage

The source problem at erdosproblems.com/1087 asks to "Estimate $f(n)$" and then poses the specific conjecture: **"is it true that $f(n) \leq n^{3+o(1)}$?"** This is a yes/no question (Case 1).

- [ ] **FAIL — Missing `answer()` wrapper.** The problem asks "Is it true that …", which is a yes/no question (Case 1 per §6). The theorem should use the form `answer(sorry) ↔ P`, not a direct assertion. Currently:
  ```lean
  theorem erdos_1087 (ε : ℝ) (hε : 0 < ε) :
      ∃ N : ℕ, ∀ n : ℕ, n ≥ N → ... := by sorry
  ```
  Should be:
  ```lean
  theorem erdos_1087 : answer(sorry) ↔
      ∀ (ε : ℝ), 0 < ε → ∃ N : ℕ, ∀ n : ℕ, n ≥ N →
        ∀ P : Finset (EuclideanSpace ℝ (Fin 2)),
          P.card = n →
          (degenerateQuadrupleCount P : ℝ) ≤ (n : ℝ) ^ ((3 : ℝ) + ε) := by
    sorry
  ```
- [ ] **FAIL — Quantifiers as theorem parameters.** `(ε : ℝ) (hε : 0 < ε)` are theorem parameters but should appear after `answer(sorry)` on the RHS of `↔`. The linter `linter.style.answer_attribute` would flag this once `answer()` is added.
