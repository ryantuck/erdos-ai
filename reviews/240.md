### `answer()` usage

- [ ] **FAIL — Missing `answer()` wrapper (Case 1 applies).** The original problem asks a yes/no question: *"Is there an infinite set of primes P such that…?"* Per the checklist, "Is there…" / "Does there exist…" triggers **Case 1**. The theorem should use `answer(True) ↔ P` since Tijdeman proved the answer is yes. Currently the theorem states the existential directly:
  ```lean
  theorem erdos_240 :
      ∃ P : Set ℕ, Set.Infinite P ∧ ... := by sorry
  ```
  It should instead be:
  ```lean
  theorem erdos_240 :
      answer(True) ↔
        ∃ P : Set ℕ, Set.Infinite P ∧ (∀ p ∈ P, Nat.Prime p) ∧
          ∀ B : ℕ, Set.Finite {n : ℕ | n ∈ smoothNumbers P ∧
            ∃ m ∈ smoothNumbers P, n < m ∧ m ≤ n + B} := by
    sorry
  ```
- [x] Quantifiers are in the theorem body, not as theorem parameters. (Correct positioning for `answer()` once added.)
