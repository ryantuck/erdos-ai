## 6. Correctness

### 6.1 The `answer(True)` wrapper

### 6.2 The independent set condition

The hypothesis states: for all `S : Finset (Fin n)`, there exists `I` with `G.IndepSetIn I S ‚ąß 2 * I.card ‚Č• S.card - k`.

**Potential issue with natural number subtraction.** The condition `2 * I.card ‚Č• S.card - k` uses natural number subtraction, which truncates to 0. This means:
- When `S.card ‚Č§ k`, the condition becomes `2 * I.card ‚Č• 0`, which is trivially true for any `I` (including the empty set).
- The mathematical statement requires |I| ‚Č• (|S| - k) / 2, which for |S| ‚Č§ k should indeed be trivially satisfied (any independent set works).

So the truncation is actually **mathematically correct** here ‚ÄĒ when the subset has at most k vertices, the condition is vacuous, which matches the intended meaning. ‚úÖ

**Comparison with Problem 922's encoding:** Problem 922 uses `2 * T.card + k ‚Č• S.card`, which is equivalent to `T.card ‚Č• (S.card - k) / 2` but avoids truncation entirely. The two encodings are equivalent over ‚Ąē:
- `2 * c ‚Č• a - b` (with ‚Ąē truncation) ‚üļ `2 * c + b ‚Č• a` (no truncation needed)

This is because for natural numbers, `x ‚Č• a - b ‚ÜĒ x + b ‚Č• a`. So the conditions are provably equivalent. The Problem 922 style is slightly more robust, but both are correct. ‚úÖ

### 6.3 The bipartiteness conclusion

The conclusion states: there exists `T : Finset (Fin n)` with `T.card ‚Č§ C` and a coloring `f : Fin n ‚Üí Bool` such that for all `u, v` not in `T`, if `G.Adj u v` then `f u ‚Č† f v`.

This correctly encodes "removing T makes G bipartite": the induced subgraph on `Fin n \ T` is 2-colorable. The function `f` is defined on all of `Fin n` but its values on `T` are unconstrained ‚ÄĒ this is fine because we only require the proper-coloring property outside `T`. ‚úÖ
