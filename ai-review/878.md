The formalization captures five conjectures:

2. F(n) ≫ n log log n for almost all n ✓
3. max_{n≤x} f(n) ~ x log x / log log x ✓
4. max_{n≤x} f(n) = max_{n≤x} F(n) for sufficiently large x ✓

**Conjecture (4) ambiguity:** The website phrases this as "for all or sufficiently large x," acknowledging uncertainty. The formalization chose "sufficiently large x" (the weaker claim), which is reasonable and arguably the intended interpretation.

**Subtleties:**
