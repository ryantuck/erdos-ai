### Indexing: $\{0, \ldots, n-1\}$ vs $\{1, \ldots, n\}$

The original problem uses $\{1, \ldots, n\}$ while the formalization uses $\text{Fin } n = \{0, \ldots, n-1\}$. The descending wave condition is translation-invariant: if $(x_1, \ldots, x_k)$ satisfies $x_j \geq (x_{j+1} + x_{j-1})/2$, then so does $(x_1 + c, \ldots, x_k + c)$ for any constant $c$. Since 2-colorings of $\{0, \ldots, n-1\}$ biject with 2-colorings of $\{1, \ldots, n\}$ (via the shift), $f(k)$ is the same under either convention. **No issue.**
