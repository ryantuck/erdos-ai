## 6. Correctness

### Definition of `minTriangleEdges`

The set `{m : ‚Ąē | ‚ąÄ G : SimpleGraph (Fin n), G.edgeSet.ncard ‚Č• m ‚Üí (r : ‚Ąē‚ąě) ‚Č§ G.chromaticNumber ‚Üí ¬¨G.CliqueFree 3}` correctly captures "the minimum m such that every n-vertex graph with ‚Č• m edges and chromatic number ‚Č• r contains a triangle."

**Key properties verified:**
- **Upward-closed**: If all graphs with ‚Č• m edges satisfy the property, then so do all graphs with ‚Č• (m+1) edges (a subset). So `sInf` correctly returns the threshold.
- **Nonempty**: For any n, taking m > n(n‚ąí1)/2 makes the hypothesis `G.edgeSet.ncard ‚Č• m` vacuously false, so the set is nonempty.
- **`sInf` on ‚Ąē**: When the set is nonempty, `sInf` returns the minimum element (‚Ąē is well-ordered). When empty, it returns 0. The set is always nonempty here, so this is fine.
- **`Fin n` for vertex type**: Correctly models "graphs on n (labeled) vertices." The chromatic number and edge count are invariant under relabeling, so using labeled vertices is mathematically equivalent.

### Theorem statement

- **`r ‚Č• 2` precondition**: Correct. For r ‚Č§ 1, the chromatic number condition is trivial and the problem is degenerate.
- **Asymptotic form**: The expression `(n : ‚ĄĚ)^2 / 4 - g_r / 2 * (n : ‚ĄĚ)` matches the Simonovits asymptotic n¬≤/4 ‚ąí g(r)n/2.
